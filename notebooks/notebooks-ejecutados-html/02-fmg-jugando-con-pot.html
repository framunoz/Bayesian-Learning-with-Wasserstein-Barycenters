Se definen los parmetros importantes de los puntos a grficar

In [3]:
n = 50  # nb samples

mu_s = np.array([0, 0])
cov_s = np.array([[1, 0], [0, 1]])

mu_t = np.array([4, 4])
cov_t = np.array([[1, -.8], [-.8, 1]])

Se definen los puntos de forma que $x_s \sim \mathcal{N}(\mu_s, \Sigma_s)$ y $x_t \sim \mathcal{N}(\mu_t, \Sigma_t)$. Adems se definen los pesos de forma uniforme. Notar que $x_s$ tiene dimensiones de $(50, 2)$, es decir, $(n_\text{data}, n_\text{dim})$.

In [4]:
xs = ot.datasets.make_2D_samples_gauss(n, mu_s, cov_s)
xt = ot.datasets.make_2D_samples_gauss(n, mu_t, cov_t)

a, b = np.ones((n,)) / n, np.ones((n,)) / n  # uniform distribution on samples

# loss matrix
M = ot.dist(xs, xt)
